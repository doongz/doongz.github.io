高斯求和

根据高斯求和公式，将从 0 到 n 的全部整数之和记为 total

total=∑i=1nnums[i]=(n+1)∗n2total = \sum_{i=1}^{n}nums[i] = \frac{(n+1)*n}{2}total=i=1∑n​nums[i]=2(n+1)∗n​

推广：任意起点，任意间距的高斯求和公式

total=∑i=1nnums[i]=(nums[start]+nums[end])∗size2total = \sum_{i=1}^{n}nums[i] = \frac{(nums[start] + nums[end]) * size}{2}total=i=1∑n​nums[i]=2(nums[start]+nums[end])∗size​

例如 [2,4,6,8]，[2,4,6]

应用：

  • 找出 0 到 n 连续数组中，缺失的那个数
#include <iostream>
#include <vector>

using namespace std;

int Gaussian(vector<int> &arr) {
    int sz = arr.size();
    return (arr[0] + arr[sz - 1]) * sz / 2;
}

int main() {
    vector<int> arr = {2, 4, 6, 8};
    int total = Gaussian(arr);
    cout << total << endl;  // 20
    return 0;
}
