循环

循环类型描述
for 循环多次执行一个语句序列，简化管理循环变量的代码。
while 循环当给定条件为真时，重复语句或语句组。它会在执行循环主体之前测试条件。
do...while 循环除了它是在循环主体结尾测试条件外，其他与 while 语句类似。
嵌套循环您可以在 while、for 或 do..while 循环内使用一个或多个循环。

一、循环控制语句​

控制语句描述
break 语句终止 loop 或 switch 语句，程序流将继续执行紧接着 loop 或 switch 的下一条语句。
continue 语句引起循环跳过主体的剩余部分，立即重新开始测试条件。
goto 语句将控制转移到被标记的语句。但是不建议在程序中使用 goto 语句。

二、实例​

for 普通使用

for (int a = 0; a < 3; a++) {
    cout << "a: " << a << endl;
}
// a: 0
// a: 1
// a: 2

for 循环取值

int arr[3] = {1,2,3};
for (int num : arr) { // 如果需要改变，声明成引用 int &num
    cout << "num: " << num << endl;
}
// num: 1
// num: 2
// num: 3

无限循环

int a = 0;
for (;;) {
    if (a >= 3) {
        break;
    }
    cout << "a: " << a << endl;
    a++;
}
// a: 0
// a: 1
// a: 2

while

int a = 0;
while (a < 3) {
    cout << "a: " << a << endl;
    a++;
}
// a: 0
// a: 1
// a: 2

do while

int a = 0;
do {
    cout << "a: " << a << endl;
    a++;
} while (a < 3);
// a: 0
// a: 1
// a: 2
